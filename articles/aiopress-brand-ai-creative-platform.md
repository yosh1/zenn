## プロジェクト概要

### 誰のためのプロダクトか

- 中小企業のマーケティング担当者や経営者
- スタートアップの少人数チーム
- ブランド発信を始めたいが、デザイナーやコピーライターを雇う余裕がない組織
- SNS、ブログ、プレゼン資料をひとりで回している担当者

### ブランドの一貫性と制作コスト、どちらも諦めたくない

> SNS投稿、ブログ記事、営業資料… 全部自分で作らなきゃいけないけど、デザインもコピーも自信がない

> ChatGPT で文章は作れるようになったけど、毎回ブランドのトーンがバラバラになる

> 外注すると1案件で数万円。月に何十本もは出せない

生成AIでテキストや画像を作ること自体は簡単になりました。ただ、ブランドとして一貫性のあるクリエイティブを継続的に大量に出し続けるのは、まだ難しいままです。

汎用AIツールだと、プロンプトを打つたびにブランドのトーンやカラー、ターゲット像を伝え直さなければなりません。出力の質も毎回ばらつきます。結局、無個性なコンテンツが量産されるか、ブランドガイドラインとの突き合わせに人手がかかるか、どちらかに陥ります。

### AIOプレスが解決すること

AIOプレスは、ブランドのDNA──ミッション、ビジョン、ターゲット像、トーン&マナー、カラーパレット、フォント──を一度登録すれば、そのDNAに沿ったクリエイティブをワンクリックで生成できるプラットフォームです。

加えて、AIエージェントを搭載しています。自然言語で指示を出すだけで、複数のツールを組み合わせながらクリエイティブの生成、評価、改善までを自動で回します。

### 3つの特徴

**1. ブランドDNAが生成を駆動する**

ブランドの構成要素を構造化データとして保持し、すべての生成プロセスに自動で差し込みます。プロンプトを毎回書き直す必要はありません。ミッション、ビジョン、ターゲット、トーン&マナーの4軸で定義したDNAが、テキストにも画像にもプレゼン資料にもそのまま反映されます。

**2. ひとつのテーマから複数フォーマットを一括生成**

| フォーマット | 生成内容 | 活用シーン |
|---|---|---|
| キャッチコピー | 3パターンのコピー案 | 広告、LP、SNS |
| SNS投稿 | プラットフォームに最適化した投稿文 | X、Instagram、LinkedIn |
| ブログ記事 | SEOを意識した長文コンテンツ | オウンドメディア |
| ブランド画像 | ブランドカラーと世界観を反映したビジュアル | バナー、サムネイル |
| プレゼン資料 | PPTX形式のスライドデッキ | 営業、社内共有 |

**3. AIエージェントが自律的にクリエイティブを作る**

ReAct パターンで実装したAIエージェントが、自然言語の指示からブランド情報の取得、過去作品の参照、テキスト生成、画像生成、ブランド適合度評価までを自律的に実行します。人間は方向性を示して、最終判断に集中すればよい形です。

## 主要機能

### 1. ブランドDNA定義

ブランドを構成する要素を登録します。

```
Brand DNA
├── Mission（存在意義）
├── Vision（目指す世界）
├── Value Proposition（価値提案）
├── Target Audience（ペルソナ）
├── Tone & Manner（語り口）
├── Personality（ブランドの人格）
├── Keywords（キーワード群）
├── Color Palette（ブランドカラー）
├── Typography（フォント体系）
└── Design Principles（デザイン原則）
```

登録したDNAはすべてのAI生成リクエストに自動で組み込まれます。URLやファイルをアップロードすれば、Gemini と Cloud Vision API が連携してブランド特性を抽出し、DNAの初期値を埋めてくれます。PDF、画像、Webページに対応しています。

### 2. AIクリエイティブ生成

Gemini 2.0 Flash でテキストを、Imagen 3 でブランドの世界観に合った画像を生成し、複数パターンをまとめて出力します。

生成時にはブランドフィットスコアを0〜100点で自動算出し、DNAとの整合度を可視化します。

| 評価軸 | 配点 |
|---|---|
| ミッション、ビジョンとの整合性 | 25点 |
| ターゲットへの適切さ | 25点 |
| トーン&マナーの反映度 | 25点 |
| 創造性、独自性 | 25点 |

スコアが低ければ改善ポイントも一緒に出ます。どこがずれているかが見えるので、次の生成に活かせます。

### 3. AIエージェント (ReAct パターン)

Gemini 2.0 Flash の Function Calling で動くAIエージェントを搭載しています。自然言語で指示を出すと、以下のツールを自分で選んで組み合わせ、タスクを遂行します。

| ツール | 機能 |
|---|---|
| `get_brand_info` | ブランドDNA情報の取得 |
| `search_past_creatives` | 過去のクリエイティブを検索 |
| `generate_text_creative` | テキストの生成 |
| `generate_image` | 画像の生成 |
| `evaluate_brand_fit` | ブランド適合度の評価 |
| `suggest_keywords` | キーワードの提案 |

Thought、Action、Observation のループを最大10回まで回し、複雑なタスクも段階的にこなします。会話履歴は Firestore に保存されるため、セッションをまたいだ対話もできます。

### 4. フィードバックによる改善

生成されたクリエイティブに対して、自然言語でフィードバックを送れます。

```
[ユーザー] もう少しカジュアルなトーンにしたい。絵文字も使って。
    ↓
[AI] ブランドDNAのトーン設定と照合しつつ、カジュアル度を上げた改善案を3パターン生成
    ↓
[ユーザー] 2番目がいい。ただし最後の一文だけ変えたい。
    ↓
[AI] 部分修正を反映した最終案を提示
```

コンテンツの特定箇所をハイライトして、ピンポイントにフィードバックを送ることもできます。丸投げではなく、AIと一緒に仕上げていく形です。

### 5. アセット管理と自動分析

ロゴや参考画像をアップロードすると、Cloud Vision API が自動で分析します。色の抽出、画像内テキストのOCR、ラベル検出、ブランド関連要素の検出まで対応しています。

### 6. プレゼン資料の自動生成

ブランドDNAに基づいて PptxGenJS で PPTX 形式のスライドを生成します。ブランドカラーやフォントが自動で反映されるので、そのまま営業資料や社内共有に使えます。

### 7. 物理出力との連携

Epson Connect API と統合し、生成したクリエイティブをそのままプリンターに送れます。ポスターやチラシなどオフラインの施策にもすぐ対応できます。

### 8. ブランドメンバー管理

OWNER、ADMIN、MEMBER の3段階でロールベースのアクセス制御を行っています。メンバー招待機能があり、チームでブランドを運用する体制に対応しています。

## 技術的なこだわり

### ReAct パターンによるエージェント実装

Gemini 2.0 Flash の Function Calling を使い、思考、行動、観察のループで複雑なタスクを段階的に解くエージェントを実装しました。ブランド情報の取得から過去作品の分析、新規クリエイティブの生成と評価まで、ユーザーの一言の指示から自律的にやり切ります。

### プロンプトにブランドの人格を埋め込む

ブランドDNAを構造化されたコンテキストとしてプロンプトに差し込む独自のテンプレートシステムを作りました。単なる指示文ではなく、ブランドがどう振る舞うかをAIに理解させることで、出力のトーンがぶれなくなります。

### セキュリティ

- Firestore Security Rules でドキュメント単位のアクセスを制御
- ブランドごとにメンバーを管理し、ロールで権限を分離
- Cloud Functions の全エンドポイントで `verifyBrandMember` による認証検証を実行

### スケーラビリティ

- サーバーレス構成で、利用量に応じて自動スケール
- Cloud Run は最大2インスタンス、同時接続80、CPU 1、メモリ512MB
- Cloud Functions はエージェント機能に合わせてタイムアウトを300秒に設定
- Firestore の複合インデックスでクエリを最適化
- Cloud Storage でアセットを管理

## 使用した Google Cloud プロダクト

### アプリケーション実行（必須要件）

- **Cloud Run** ── Firebase App Hosting 経由で Next.js をホスティング
- **Cloud Functions** ── クリエイティブ生成、画像解析、エージェント処理、フィードバック処理のバックエンドAPI

### AI技術（必須要件）

- **Vertex AI - Gemini 2.0 Flash** ── テキスト生成、ブランドフィットスコア算出、フィードバック分析、AIエージェントの推論エンジン
- **Vertex AI - Imagen 3** ── ブランドの世界観を反映した画像生成
- **Cloud Vision API** ── アップロードされたアセットの色分析、OCR、ラベル検出

### その他の Firebase プロダクト

- **Firebase Authentication** ── Google OAuth とメールパスワードによるユーザー認証
- **Cloud Firestore** ── ブランドDNA、クリエイティブ、フィードバック、エージェントセッションの永続化
- **Cloud Storage for Firebase** ── アセットファイルと生成画像の保存、配信

## おわりに

AIOプレスは、ブランドの一貫性を保ちながら、専門知識がなくてもクリエイティブを量産できるプラットフォームです。Vertex AI の Gemini と Imagen に、ReAct パターンのAIエージェントを組み合わせ、テキストと画像の両面からブランドらしさを維持した制作物を誰でも作れるようにしました。

AIに仕事を奪われるのではなく、AIとブランドを一緒に育てる。AIOプレスはそこを目指しています。
